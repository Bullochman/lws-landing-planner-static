# lws-landing-planner-static

Static-only build of [LWS Landing Planner](https://github.com/Bullochman/lws-landing-planner) for GH Pages.

**Live at:** https://bullochman.github.io/lws-landing-planner-static/

`fake-server.js` ports server.py v0.3's full rank-tier ring-assignment algorithm to JS (R5+titled-R4 → Ring 1 rally-leader, untitled R4 → Ring 2 defender, R3 → Ring 3 supporter, R2/R1 → Ring 4+ hunter; Ministry-gate warnings; misplaced-leader warnings; freeze-risk annotations). Client code unchanged.
